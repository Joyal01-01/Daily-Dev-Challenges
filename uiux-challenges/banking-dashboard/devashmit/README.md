# 🎨 Banking Dashboard UI — Day 2 UI/UX Challenge

**Issue:** [#217](https://github.com/abhishek-goswami1/Daily-Dev-Challenges/issues/217) | Week 1 | Intermediate

## 📋 Description

A responsive finance dashboard UI featuring account balance cards, recent transactions list, a spending chart (pure CSS), and quick action buttons. Includes both light and dark mode with WCAG AA compliant contrast ratios.

## ✨ Features

- Account balance card with masked card number
- Recent transactions list (green = income, red = expense)
- Spending breakdown chart (CSS-based bar chart)
- Quick action buttons (Transfer, Pay, Top Up, More)
- Light / Dark mode toggle
- Fully responsive layout
- WCAG AA contrast ratios throughout

## 🧠 Concepts Practiced

- Data visualization with CSS
- Accessibility and color semantics
- Card layouts and grid systems
- Dark/light mode theming with CSS variables

## 🚀 How to Run

Open `index.html` directly in any browser — no build step needed.
