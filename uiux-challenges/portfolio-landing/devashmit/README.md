# 🎨 Personal Portfolio Landing Page — Day 1 UI/UX Challenge

**Issue:** [#208](https://github.com/abhishek-goswami1/Daily-Dev-Challenges/issues/208) | Week 1 | Beginner

## 📋 Description

A polished, fully responsive portfolio landing page with hero, about, skills, projects grid, and contact sections. Built with an 8px spacing system, 2-font max, and tested for real mobile.

## ✨ Features

- Hero section with CTA
- About section
- Skills grid with icons
- Projects grid with cards
- Contact section with links
- Fully responsive (mobile-first)
- 8px spacing system throughout
- Clean visual hierarchy

## 🧠 Concepts Practiced

`Typography scale` · `8px grid` · `Responsive layout` · `Visual hierarchy`

## 🚀 How to Run

Open `index.html` directly in any browser — no build step needed.
