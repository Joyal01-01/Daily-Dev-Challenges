# 🎨 Mobile App UI — Habit Tracker — Day 3 UI/UX Challenge

**Issue:** [#229](https://github.com/abhishek-goswami1/Daily-Dev-Challenges/issues/229) | Week 1 | Intermediate

## 📋 Description

A mobile-first (375px) habit tracker UI featuring a daily checklist, weekly streak calendar, SVG progress rings per habit, and a bottom navigation bar. All tap targets are minimum 44×44px.

## ✨ Features

- Daily habit checklist with tap-to-complete
- Weekly streak calendar (7-day view)
- SVG progress ring per habit showing completion %
- Bottom navigation bar (Today, Stats, Settings)
- Mobile-first at 375px, scales up gracefully
- Minimum 44×44px touch targets throughout

## 🧠 Concepts Practiced

`Mobile-first design` · `Touch targets` · `SVG progress rings` · `Bottom nav`

## 🚀 How to Run

Open `index.html` in any browser. Best viewed at 375px width (mobile).
