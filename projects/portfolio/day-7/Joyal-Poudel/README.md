# Day 7: Deployment & Final Polish 🚀

Welcome to the **final phase** of the 7-day portfolio build! This project represents a fully responsive, modern web portfolio crafted with React, GSAP for rich animations, and Tailwind CSS for utility-first styling.

## 🌟 What's New in Day 7?
- **Premium Footer:** A fully built-out dynamic footer with social links, built-in branding, and navigation.
- **Scroll-to-Top Button:** Enhancing User Experience with a smart, auto-revealing scroll-to-top button featuring an SVG circular progress indicator.
- **Accessibility Improvements:** Added native `skip-link` functionality.
- **SEO Optimization:** Comprehensive meta-tags added including Open Graph image attributes, Twitter Card data, and structured JSON-LD data points. Added `robots.txt` and `sitemap.xml`.
- **Polish & Performance:** Hover animations refined.

## 🛠 Tech Stack
- Frontend: React (Vite)
- Language: JavaScript (ES6+)
- Styling: Tailwind CSS & Custom CSS Variables (Glassmorphism design sequence)
- Animation: GSAP & GSAP ScrollTrigger
- Theming: Custom Hook for Dark/Light Mode + LocalStorage Persistence
- Deployment Ready: Vercel / Netlify optimized

## 🏃‍♂️ Getting Started Locally

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Run a production build
npm run build
```
