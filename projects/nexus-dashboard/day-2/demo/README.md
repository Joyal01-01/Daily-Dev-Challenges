# Nexus Dashboard - Phase 2: Real-time Weather Widget

## 🎯 Objective
Integrate the OpenWeatherMap API. Fetch data based on user location with graceful error handling and loading states.

## ✅ Completed Features
- ✅ Geolocation API integration
- ✅ OpenWeatherMap API integration
- ✅ Real-time weather data fetching
- ✅ Loading skeleton states
- ✅ Error state handling
- ✅ Temperature, humidity, wind speed display
- ✅ Weather description and "feels like" temperature

## 🚀 Getting Started
```bash
npm install
npm run dev
npm run build
```
