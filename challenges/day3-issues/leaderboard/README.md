# Leaderboard: Interactive Rankings Interface

## 📋 Description
An interactive leaderboard displaying top contributors with animated podium for top 3 and a comprehensive rankings table with smooth entrance animations.

## ✨ Features
- **Top 3 Podium**: Visual display of 1st, 2nd, and 3rd place with medals
- **Rankings Table**: Detailed table showing rank, user, points, and challenges solved
- **Animated Entrance**: Staggered animations for podium and table rows
- **User Avatars**: Profile images for each participant
- **Status Indicators**: Visual status display for achievements
- **Points Display**: Total points earned by each user
- **Challenge Count**: Number of challenges completed
- **Responsive Design**: Works on all screen sizes

## 🛠️ Technologies Used
- **HTML**: Leaderboard structure with table markup
- **CSS**: Podium styling and animations
- **JavaScript**: Staggered animation timing

## 📁 Files
- `index.html` - Leaderboard markup with podium and table
- `styles.css` - Podium styling and animation effects
- `script.js` - Staggered animation logic

## 🚀 How to Use
1. Open `index.html` in a web browser
2. Watch the top 3 podium animate into view
3. See the rankings table animate with staggered timing
4. View user avatars and statistics
5. Check points and challenge completion counts

## 🎨 Animation Details
- Podium items: Staggered entrance from center
- Table rows: Sequential fade-in animation
- Icons: Medal emojis for visual hierarchy
- Smooth transitions for polish

## 📊 Data Display
- **Top 3 Podium**:
  - 1st Place: Highest points
  - 2nd Place: Second highest
  - 3rd Place: Third highest

- **Rankings Table**:
  - Rank position
  - User name
  - Total points
  - Challenges completed
  - Achievement status
