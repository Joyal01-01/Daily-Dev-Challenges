# Hard Task 5: Advanced Search Bar with Filtering

## 📋 Description
Advanced search functionality featuring real-time filtering, dynamic suggestions, category filtering, and search result highlighting. Demonstrates complex JavaScript interactions and data manipulation.

## ✨ Features
- **Real-time Search**: Filter results as user types
- **Search Suggestions**: Dynamic dropdown with matching suggestions
- **Category Filtering**: Filter by product categories
- **Result Highlighting**: Matched text highlighted in search results
- **Suggestion Dropdown**: Click suggestions to populate search
- **Multiple Result Types**: Users, products, categories
- **Search Data**: Pre-populated search database
- **No Results Handling**: Shows message when no matches found

## 🛠️ Technologies Used
- **HTML**: Search bar and results container
- **CSS**: Styling for search interface
- **JavaScript**: Search algorithm, filtering, and highlighting logic

## 📁 Files
- `index.html` - Search interface markup
- `styles.css` - Search bar and results styling
- `script.js` - Search logic with filtering and highlighting

## 🚀 How to Use
1. Open `index.html` in a web browser
2. Type in the search input field
3. View suggestions appearing in dropdown
4. Click on any suggestion to populate search
5. See filtered results below
6. Matched text is highlighted in results

## 🔍 Search Functions
- `performSearch()`: Filters search data based on input
- `showSuggestions()`: Displays matching suggestions
- `highlightMatch()`: Highlights matched text with `<strong>` tags
- `updateResults()`: Renders search results to DOM

## 📊 Search Data Structure
Search data includes:
- Product information
- User profiles
- Category tags
- Various metadata

## 🎯 Features
- Case-insensitive searching
- Regex-based text highlighting
- Smart suggestion ranking
- Multiple result categories
