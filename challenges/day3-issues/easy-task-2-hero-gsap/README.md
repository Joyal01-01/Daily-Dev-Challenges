# Easy Task 2: Hero Section with GSAP Animations

## 📋 Description
A stunning hero section featuring animated entrance effects, floating shapes, and smooth scroll interactions using GSAP (GreenSock Animation Platform).

## ✨ Features
- **GSAP Animations**: Professional animation library for smooth transitions
- **Hero Title Animation**: Staggered text animation on page load
- **Floating Shapes**: Continuous animated shapes with yoyo effect
- **Call-to-Action Button**: Interactive button with hover effects
- **Navbar Scroll Effect**: Background changes when scrolling
- **Feature Cards**: Staggered entrance animation with IntersectionObserver
- **Responsive Design**: Works seamlessly on all device sizes

## 🛠️ Technologies Used
- **HTML**: Semantic markup with hero section
- **CSS**: Modern styling with transitions
- **JavaScript**: GSAP integration and scroll event handling
- **GSAP Library**: 3.12.2 (via CDN)

## 📁 Files
- `index.html` - HTML structure with navbar, hero, and features section
- `styles.css` - Responsive styling and layout
- `script.js` - GSAP animations and scroll interactions

## 🚀 How to Use
1. Open `index.html` in a web browser
2. Watch the animated hero section entrance
3. Scroll down to see feature cards animate into view
4. Notice navbar background changes on scroll

## 📦 Dependencies
- GSAP 3.12.2 (loaded via CDN in HTML)
