# Medium Task 4: Profile Card UI

## 📋 Description
Beautiful profile card component displaying user information with interactive follow button, responsive grid layout, and smooth animations.

## ✨ Features
- **Profile Avatar**: User profile image with hover effects
- **User Information**: Display name, role, and bio
- **Statistics Display**: Followers, following, and posts count
- **Follow Button**: Toggle follow/unfollow state with visual feedback
- **Cover Image**: Decorative header image for each card
- **Responsive Grid**: Adapts to different screen sizes
- **Hover Animations**: Smooth transitions on hover
- **Staggered Animation**: Cards animate into view with timing offset

## 🛠️ Technologies Used
- **HTML**: Card component structure
- **CSS**: Grid layout and hover effects
- **JavaScript**: Follow button toggle functionality

## 📁 Files
- `index.html` - Profile card grid with sample user data
- `styles.css` - Card styling and animations
- `script.js` - Follow button interaction logic

## 🚀 How to Use
1. Open `index.html` in a web browser
2. Browse the profile cards in the grid
3. Hover over cards to see animation effects
4. Click the "Follow" button to toggle follow state
5. Button changes to "Unfollow" when followed

## 🎨 Customization
- Replace avatar URLs with different images
- Modify user information in HTML
- Adjust grid columns for different layouts
- Customize colors and hover effects in CSS

## 📱 Responsive Breakpoints
- Mobile: 1 column
- Tablet: 2 columns
- Desktop: 3+ columns
