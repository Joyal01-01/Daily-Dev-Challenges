# Medium Task 3: Dropdown Menu with Keyboard Navigation

## 📋 Description
A fully functional dropdown menu system with keyboard support, accessibility features, and smooth animations. Includes multiple dropdown levels and auto-close functionality.

## ✨ Features
- **Click Handlers**: Toggle dropdown visibility on click
- **Keyboard Navigation**: 
  - ArrowUp/ArrowDown: Navigate menu items
  - Enter/Space: Select menu item
  - Escape: Close dropdown
  - Tab: Move to next item
- **Auto-close on Outside Click**: Dropdown closes when clicking outside
- **Focus Management**: Proper focus handling for accessibility
- **Smooth Animations**: CSS transitions for dropdown appearance
- **Multiple Dropdowns**: Support for multiple dropdown menus
- **Dividers**: Visual separators in menu items

## 🛠️ Technologies Used
- **HTML**: Semantic navigation markup
- **CSS**: Styling with smooth transitions
- **JavaScript**: Event delegation and keyboard handling

## 📁 Files
- `index.html` - Navigation structure with dropdowns
- `styles.css` - Responsive menu styling
- `script.js` - Keyboard and click event handlers

## 🚀 How to Use
1. Open `index.html` in a web browser
2. Click on menu items with dropdown arrow to toggle
3. Use keyboard arrows to navigate items
4. Press Enter or Space to select
5. Press Escape to close dropdown
6. Click outside to auto-close

## ✅ Accessibility
- Full keyboard navigation support
- ARIA labels for screen readers
- Focus indicators on all interactive elements
- Semantic HTML structure

## ✅ Status
Complete and fully functional with accessibility support
