# Easy Task 1: Form Validation

## 📋 Description
A comprehensive form validation system with real-time error checking and validation rules for various input types.

## ✨ Features
- **Real-time Validation**: Validates input as user types
- **Multiple Field Types**: Username, email, password, phone number validation
- **Custom Validation Rules**:
  - Username: 3-20 characters, alphanumeric
  - Email: Valid email format
  - Password: Minimum 8 characters, requires uppercase and numbers
  - Phone: Valid phone number format
- **Visual Feedback**: Color-coded error messages
- **Form Submit**: Success message on valid submission

## 🛠️ Technologies Used
- **HTML**: Form structure and semantic markup
- **CSS**: Styling and form design
- **JavaScript**: Real-time validation logic

## 📁 Files
- `index.html` - Form markup with input fields and error containers
- `styles.css` - Styling for form layout and error states
- `script.js` - Validation logic with event handlers

## 🚀 How to Use
1. Open `index.html` in a web browser
2. Fill in the form fields
3. Error messages appear instantly as you type
4. Submit form when all fields are valid
