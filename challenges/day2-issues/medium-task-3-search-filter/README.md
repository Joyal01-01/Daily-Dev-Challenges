# Medium Task 3: Search Filter

## 📋 Description
A real-time product search and filtering system with dynamic result counting and responsive grid display.

## ✨ Features
- **Real-time Search**: Filter products as user types
- **Dynamic Result Counter**: Shows number of matching products
- **Product Grid Display**: Responsive layout for products
- **Search History**: Remembers previous searches
- **No Results Handling**: Shows message when no products match
- **Category Tags**: Organize products by type
- **Product Cards**: Display product details with images
- **Case-Insensitive Search**: Search works regardless of capitalization

## 🛠️ Technologies Used
- **HTML**: Search input and product grid structure
- **CSS**: Product card styling and responsive layout
- **JavaScript**: Real-time filtering algorithm

## 📁 Files
- `index.html` - Search interface and product grid markup
- `styles.css` - Product card styling and layout
- `script.js` - Search filtering logic

## 🚀 How to Use
1. Open `index.html` in a web browser
2. Type in the search input to filter products
3. Watch result count update in real-time
4. Clear search to see all products again
5. View filtered products in the grid below

## 🔍 Search Capabilities
- Filter by product name
- Filter by category
- Filter by price range
- Multiple keyword search
- Partial word matching

## 📊 Product Data
The application includes sample products with:
- Product name
- Category tag
- Price information
- Product image/icon
- Description

## 📱 Responsive Design
- **Mobile**: Single column product grid
- **Tablet**: 2 column layout
- **Desktop**: 3+ column layout
- **Large Screens**: Full width optimization
