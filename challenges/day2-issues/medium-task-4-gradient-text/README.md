# Medium Task 4: Gradient Text Effects

## 📋 Description
A showcase of various CSS gradient text effects with animations, demonstrating multiple techniques for creating visually stunning text.

## ✨ Features
- **Static Gradient**: Simple gradient applied to text
- **Animated Gradient**: Smoothly flowing gradient animation
- **Rainbow Text**: Multi-color gradient effect
- **Outlined Gradient**: Gradient stroke on text outline
- **Pulse Effect**: Gradient that pulses with light
- **Shift Effect**: Gradient color shifting animation
- **Wave Effect**: Undulating gradient animation
- **Neon Glow**: Glowing gradient text effect

## 🛠️ Technologies Used
- **HTML**: Semantic markup for text effects
- **CSS**: Gradient syntax and animations
- **CSS Animations**: Keyframes for smooth transitions
- **CSS Gradients**: Linear and radial gradients

## 📁 Files
- `index.html` - Text effects showcase markup
- `styles.css` - Gradient definitions and animations
- `script.js` - Optional interactive enhancements

## 🚀 How to Use
1. Open `index.html` in a web browser
2. Scroll to see all gradient text effects
3. Watch animations loop continuously
4. Resize window to see responsive text sizing
5. Observe different gradient techniques

## 🎨 Gradient Techniques
- **Background Gradient**: Color gradient as text background
- **Clip Path**: Text clipped to gradient shape
- **Text Stroke**: Gradient applied to text outline
- **Animation Keyframes**: Smooth color transitions
- **Multiple Layers**: Overlapping effects for depth

## 📝 Effect Details
Each effect card includes:
- **Title**: Name of the effect
- **Example Text**: The gradient text to display
- **Description**: Explanation of the technique used
