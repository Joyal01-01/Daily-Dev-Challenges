# Easy Task 2: Animated Cards

## 📋 Description
A collection of beautifully animated cards in a responsive grid layout with hover effects and interactive buttons.

## ✨ Features
- **Responsive Grid Layout**: Auto-adjusts columns based on screen size
- **Hover Animations**: Smooth scale and shadow effects
- **Icon Display**: Emoji icons for visual interest
- **Card Content**: Title, description, and call-to-action button
- **Button Interactions**: Interactive "Learn More" buttons
- **Staggered Animation**: Cards animate into view with timing offset
- **Smooth Transitions**: All effects use smooth CSS transitions
- **Modern Design**: Clean, minimal card design

## 🛠️ Technologies Used
- **HTML**: Card component structure with semantic markup
- **CSS**: Grid layout and hover animations
- **JavaScript**: Optional interactive enhancements

## 📁 Files
- `index.html` - Card grid markup with card components
- `styles.css` - Card styling and animation effects
- `script.js` - Interactive functionality

## 🚀 How to Use
1. Open `index.html` in a web browser
2. Hover over cards to see animations
3. Click "Learn More" buttons for interactions
4. Resize window to see responsive grid adjustment
5. Scroll to see staggered animation effects

## 🎨 Card Features
- **Icon**: Large emoji icon at top
- **Title**: Card heading
- **Description**: Short descriptive text
- **Button**: Interactive call-to-action button
- **Hover Effects**: Scale up, shadow enhancement

## 📱 Responsive Breakpoints
- **Mobile**: 1 column
- **Tablet**: 2 columns
- **Desktop**: 3 columns
- **Large Screens**: 4+ columns
