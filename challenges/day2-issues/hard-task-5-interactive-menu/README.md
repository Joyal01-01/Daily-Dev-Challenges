# Hard Task 5: Interactive Menu System

## 📋 Description
A sophisticated interactive menu system with keyboard navigation, submenu support, accessibility features, and multiple interaction patterns.

## ✨ Features
- **Keyboard Navigation**: Full keyboard support with arrow keys
- **Nested Submenus**: Multi-level menu hierarchy
- **Submenu Toggle**: Click to expand/collapse submenus
- **Accessibility**: ARIA labels and semantic HTML
- **Menu Trigger Button**: Hamburger-style trigger button
- **Focus Management**: Proper focus handling for keyboard users
- **Hover Effects**: Visual feedback on hover
- **Auto-close**: Submenus close on blur or escape
- **Icon Display**: Icons for menu items

## 🛠️ Technologies Used
- **HTML**: Semantic menu structure with ARIA attributes
- **CSS**: Menu styling and submenu animations
- **JavaScript**: Complex keyboard and mouse event handling

## 📁 Files
- `index.html` - Menu structure with semantic markup
- `styles.css` - Menu styling and animations
- `script.js` - Keyboard navigation and interaction logic

## 🚀 How to Use
1. Open `index.html` in a web browser
2. Click the "Menu" button to open/close menu
3. Click menu items to select them
4. Use keyboard:
   - **Arrow Up/Down**: Navigate menu items
   - **Arrow Right**: Open submenu
   - **Arrow Left**: Close submenu
   - **Enter/Space**: Select item
   - **Escape**: Close menu
5. Tab through menu items

## ⌨️ Keyboard Shortcuts
- `ArrowUp`: Move to previous menu item
- `ArrowDown`: Move to next menu item
- `ArrowRight`: Open submenu
- `ArrowLeft`: Close submenu
- `Enter` or `Space`: Select item
- `Escape`: Close menu
- `Tab`: Navigate to next focusable element

## ♿ Accessibility Features
- ARIA labels for screen readers
- Semantic HTML structure
- Keyboard-navigable
- Focus visible indicators
- Proper role attributes
- Submenu indicators
- Status announcements

## 🎯 Menu Structure
- **Main Menu**: Trigger button + menu container
- **Menu Items**: Home, About, Services, Contact, etc.
- **Submenus**: Nested items under parent menu items
- **Icons**: Visual indicators for menu items

## 📱 Responsive Design
- Desktop: Full menu display
- Mobile: Hamburger menu trigger
- Touch-friendly targets

Complete and fully functional with accessibility support
