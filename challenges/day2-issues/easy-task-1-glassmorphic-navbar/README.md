# Easy Task 1: Glassmorphic Navbar

## 📋 Description
A modern, responsive navbar featuring the popular glassmorphic design pattern with smooth animations and a mobile hamburger menu.

## ✨ Features
- **Glassmorphic Design**: Semi-transparent frosted glass effect navbar
- **Responsive Navigation**: Adapts to all screen sizes
- **Hamburger Menu**: Mobile-friendly menu toggle
- **Active State Indicators**: Visual indication of current page
- **Call-to-Action Button**: Prominent "Get Started" button
- **Smooth Animations**: Transitions and hover effects
- **Modern Styling**: Uses backdrop blur and transparency effects
- **Hero Section**: Integrated hero section below navbar

## 🛠️ Technologies Used
- **HTML**: Semantic navbar and hero markup
- **CSS**: Glassmorphic effects with backdrop blur
- **JavaScript**: Menu toggle and responsive behavior

## 📁 Files
- `index.html` - Navbar and hero section structure
- `styles.css` - Glassmorphic styling and animations
- `script.js` - Menu toggle and interaction logic

## 🚀 How to Use
1. Open `index.html` in a web browser
2. Resize window to see responsive behavior
3. Click hamburger menu on mobile to toggle navigation
4. Hover over menu items to see effects
5. Click "Explore More" to see interactive buttons

## 📱 Responsive Design
- **Desktop**: Full horizontal navigation menu
- **Tablet**: Responsive layout adjustments
- **Mobile**: Hamburger menu with vertical navigation

## 🎨 Design Pattern
Glassmorphism features:
- Backdrop blur effect
- Semi-transparent background
- Border accent
- Layered depth effect
