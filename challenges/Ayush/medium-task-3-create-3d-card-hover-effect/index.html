Implementation Guide

CSS 3D Perspective Setup

/* Container for 3D perspective */
.card-container {
    perspective: 1000px;
}

/* Card rotation based on mouse position */
.card {
    transform-style: preserve-3d;
    transition: transform 0.1s;
}

/* Calculate rotation:
   rotateX = (mouse-y - card-center-y) / card-height * rotation-amount
   rotateY = (mouse-x - card-center-x) / card-width * -rotation-amount
*/

JavaScript Mouse Tracking

card.addEventListener('mousemove', (e) => {
    const rect = card.getBoundingClientRect();
    const centerX = rect.left + rect.width / 2;
    const centerY = rect.top + rect.height / 2;
    
    const rotateX = ((e.clientY - centerY) / rect.height) * 10;
    const rotateY = ((e.clientX - centerX) / rect.width) * -10;
    
    card.style.transform = 
        `rotateX(${rotateX}deg) rotateY(${rotateY}deg)`;
});

Tips & Best Practices

⚡ Performance

Use transform and opacity for animations. Avoid layout-triggering properties like width, height.

🎯 Precision

Calculate mouse position relative to card center for accurate rotation values.

📱 Responsive

Test on mobile devices. Consider touch events for interactive features.

♿ Accessibility

Ensure keyboard navigation works. Provide alternative information display methods.
