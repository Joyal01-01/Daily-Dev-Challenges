# Day 7 Challenges

This directory contains implementations for the seven Day 7 challenges from the Daily‑Dev‑Challenges repository.

| Challenge | Directory |
|-----------|-----------|
| Full‑stack Notes App (React + FastAPI) | `fullstack-notes-app/` |
| Vanilla JS Todo App | `vanilla-todo/` |
| Multi‑threaded Port Scanner | `port-scanner/` |
| JWT Authentication System | `jwt-auth/` |
| Banking Dashboard UI | `banking-dashboard/` |
| Web Scraper with BeautifulSoup | `web-scraper/` |
| Command Palette (React) | `command-palette/` |

Each sub‑folder contains a minimal implementation and a README with usage instructions.
