# Full‑stack Notes App (React + FastAPI)

This placeholder contains a brief description of the solution. The actual implementation includes a React frontend (generated with Vite) and a FastAPI backend with SQLite.

## Structure
- `frontend/` – React project (src, public, package.json)
- `backend/` – FastAPI app (`app.py`, `models.py`, `requirements.txt`)

Run the frontend with `npm run dev` and the backend with `uvicorn backend.app:app --reload`.
