# 🟨 Real-time Search Filter with Highlighting — Day 3 JavaScript Challenge

**Issue:** [#232](https://github.com/abhishek-goswami1/Daily-Dev-Challenges/issues/232) | Week 1 | Beginner

## 📋 Description

A searchable list of 50+ countries with real-time filtering as the user types. Matched text is highlighted using regex + `<mark>` tags. Includes category filter chips (by continent) and debounced input.

## ✨ Features

- 50+ countries with continent categories
- Real-time filter as you type (debounced with `setTimeout`)
- Matched text highlighted with `<mark>` tag via regex
- Category filter chips (All, Asia, Europe, Americas, Africa, Oceania)
- Result count display
- No results state

## 🧠 Concepts Practiced

`Debouncing` · `Array filter/map` · `Regex highlight` · `Filter chips`

## 🚀 How to Run

Open `index.html` directly in any browser — no build step needed.
