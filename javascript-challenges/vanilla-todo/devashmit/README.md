# 🟨 Vanilla JS Todo App — Day 1 JavaScript Challenge

**Issue:** [#211](https://github.com/abhishek-goswami1/Daily-Dev-Challenges/issues/211) | Week 1 | Beginner

## 📋 Description

A zero-framework todo app built with pure Vanilla JavaScript. Features add, delete, mark complete, filter by status, and localStorage persistence.

## ✨ Features

- Add new todos
- Delete todos
- Mark todos as complete/incomplete
- Filter by: All / Active / Completed
- Persists to `localStorage` (survives page refresh)
- Event delegation on list container
- Item count display

## 🧠 Concepts Practiced

`DOM manipulation` · `localStorage` · `Event delegation` · `JSON.parse/stringify`

## 🚀 How to Run

Open `index.html` directly in any browser — no build step needed.
